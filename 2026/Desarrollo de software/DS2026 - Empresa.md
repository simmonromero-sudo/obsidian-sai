
Empresa de cargar o forwarder.
Algo como un despegar pero para el transporte del contenedores, en vez de aereolinas, armadores. Son intermediarios, compran un espacio del buque (alocation) (para cada armador individualmente), se llena con un contenedor y se vende a terceros, tanto importación (inpo) como exportación(expo). 
Cada armador tiene su diferentes puestos para sus buques. 

cuando se ingresa en numero de reserva y el armador otorga un numero de booking (4 letras, 6 números y un digito control, las letras o números no necesariamente identifican a cada armador), cada armador o dueño de buque o contenedores, tiene su web tracker. 
Una reserva puede tener uno o x cantidad de contenedores (todos con el mismo booking)

El único dato que aparece sobre el container es el numero de container y el tamaño.
Hay empresas que alquilan contenedores a otras empresas, estos tienen un numero de booking diferente, con los prefijos específicos de  la empresa prestadora.


1- se requiere un buque
2- se hace la reserva
3- se envía al armador
4- devuelve una orden de retiro de contenedor
5- esa orden se entrega al cliente
6- el cliente retira
7- lo carga
8- lo devuelve lleno al puerto
9- se va el buque (empieza el trackeo de la carga) 
10- El contenedor aparece como retirado en sistema
11- Cuando se devuelve aparece como entregado
12- Cuando sale el buque aparece como embarcado (fecha de embarque y fecha estimada de arribo)
13- Una vez que llega figura como arribado
14- Luego cuando el cliente en destino lo descarga 
15- por ultimo entregado (figura como entregado)
